## Goal

R4은 "실앱 손맛 / 일관성 / 접근성"을 닫는다. R1–R3에서 기능은 섰지만 남은 감(feel)·a11y·일관성 부채와 R1–R3에서 미룬 마이너 회귀를 전부 제거한다:

1. **전환·햅틱(feel)**: 탭/라우트 전환 애니(View Transitions 우선, 미지원 시 CSS 키드 크로스페이드, `prefers-reduced-motion` 존중), 활성 탭 누름 피드백, feature-detect 햅틱(저장·하트·마커드롭·방문 토글 — 시각 피드백과 항상 병행).
2. **스켈레톤·토큰 일관성**: 탭별 `isLoading`에서 한 가지 스켈레톤 컨벤션(캘린더 그리드/추천 카드), 타입 스케일 램프(`--fs-h2`/`--fs-label`), 하단 오프셋 토큰 단일화(`--tabbar-h`/`--sheet-peek-h`, `TABBAR_H` 리터럴 제거), 다크 클러스터 대비.
3. **지도 폴리시**: 클러스터 탭 `fitBounds(member bounds)`, 선택핀 `transform-origin: 50% 100%`(팁 고정), 미리보기핀 앵커, 시트 드래그 중 진행형 백드롭 딤, MapPage `?place=` 수신(setSelectedId + pan/fit → 캘린더 아젠다 장소칩 딥링크 닫기).
4. **a11y**: 트랙 범례 색+이름칩 명료화 + 타임라인 이벤트에 트랙 aria-label, 마커 비액션 라벨 aria-hidden 일관, 타입 램프 적용, **마커 키보드 조작성**(마커를 포커스·Enter/Space 활성화 가능하게, 마우스/터치 햅틱 경로와 동등 — spec line 54, Task 17). spec line 54의 "disconnect 진짜 모달(focus trap) — 공용 Dialog 프리미티브 추출"은 **이미 구현돼 있어 R4 신규 태스크가 없음**(`src/components/common/Dialog.tsx` 프리미티브 + `UsPage`가 `DisconnectConfirm`을 그 Dialog로 래핑 — 정직성 섹션 참조).
5. **이월 마이너**: WeekStrip 실제 DOW, upcomingFeed 임박창 soft 라벨, 이벤트별 타임존 표시(로컬시각 라벨/토글), offlineExecutor `${table}.delete`/`${table}.restore` 일반화(R3 T18 재생 갭), 연결해제 전 내보내기 게이트를 ZIP에만 충족 + `useUpdateProfile`의 `myProfile` 무효화, RecommendPage retro/SEED 탭 가능 or "곧 제공".

## Architecture

- 라우트 시맨틱 그대로: `createBrowserRouter` 데이터 라우터(`src/app/router.tsx`), 탭은 `@/app/tabs`의 `TABS` 단일 출처에서 도출. 전환 시맨틱은 `AppLayout`의 `<Outlet/>` 시맨틱(크로스페이드) + `TabBar` `NavLink viewTransition` prop에 얹는다.
- 모션은 CSS 토큰(`--motion-base`/`--ease`)으로만 — `prefers-reduced-motion`은 `tokens.css`가 토큰→0 + 전역 `!important` 캡으로 자동 무력화(CSS 경로). JS 경로(View Transitions 게이트)는 `matchMedia` 헬퍼로 명시 분기.
- 순수 헬퍼 우선: DOW/타임존 포맷/오프셋 계산/딥링크 선택/클러스터 바운즈 도출을 DOM-free 순수 함수로 추출해 vitest로 못박고, 컴포넌트는 그 결과만 소비.
- 햅틱은 UI 핸들러(onSuccess/onClick)에서 한 번 — mutationFn 안이 아니라 실제 성공에서만, 기존 토스트/마커/하트 시각 피드백과 항상 병행. iOS Safari `navigator.vibrate` 미지원 → feature-detect no-op.
- offlineExecutor 일반화: R3에서 `useTrash`가 6개 TrashKind(`places|events|visits|photos|trips|itineraries`)에 대해 `${table}.delete`/`${table}.restore`를 enqueue하도록 일반화됐는데(useTrash.ts L11 `TrashKind` 확인), `offlineExecutor`의 `OutboxKind`에는 `place.delete | place.restore | event.restore`만 명시돼 있고 `event.delete`·나머지 4개 테이블의 delete/restore는 **전무**(switch default가 무음 'ok'로 드롭 — 유실 0 위반). Task 21에서 명시 케이스를 정리하고 TRASH_TABLES allowlist + 정규식 default로 6×2를 전부 처리한다(테이블 allowlist로 안전).

// reduce-motion 사용자 선호(JS 경로 전용 게이트). CSS 경로는 tokens.css가 토큰→0으로 자동 처리.
export function prefersReducedMotion(): boolean {
  if (typeof window === 'undefined' || typeof window.matchMedia !== 'function') return false
  try {
    return window.matchMedia('(prefers-reduced-motion: reduce)').matches
  } catch {
    return false
  }
}
```

// feature-detect 햅틱. iOS Safari는 navigator.vibrate 미지원 → no-op. 시각 피드백과 항상 병행(ux §1).
export function haptic(pattern: number | number[] = 10): boolean {
  if (typeof navigator === 'undefined') return false
  const v = (navigator as Navigator & { vibrate?: (p: number | number[]) => boolean }).vibrate
  if (typeof v !== 'function') return false
  try {
    return v.call(navigator, pattern)
  } catch {
    return false
  }
}
```

import type { ClusterPoint } from './clusterPlaces'

export function clusterMemberPts(ids: string[], pts: ClusterPoint[]): ClusterPoint[] {
  const set = new Set(ids)
  return pts.filter((p) => set.has(p.id))
}

// 멤버 좌표 스팬이 minDeg 미만이면 degenerate(동일/근접 좌표) → fitBounds 과확대 방지.
export function boundsSpanTiny(memberPts: ClusterPoint[], minDeg = 0.0005): boolean {
  if (memberPts.length === 0) return true
  let minLat = Infinity, maxLat = -Infinity, minLng = Infinity, maxLng = -Infinity
  for (const p of memberPts) {
    if (p.lat < minLat) minLat = p.lat
    if (p.lat > maxLat) maxLat = p.lat
    if (p.lng < minLng) minLng = p.lng
    if (p.lng > maxLng) maxLng = p.lng
  }
  return (maxLat - minLat) < minDeg && (maxLng - minLng) < minDeg
}
